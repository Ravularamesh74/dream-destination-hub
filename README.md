# 🚖 Mallikarjuna Travels – Premium Cab Booking Platform

A modern, responsive, and enterprise-ready cab booking website designed for **Mallikarjuna Travels**, offering seamless travel solutions including outstation trips, airport transfers, city tours, and pilgrimage packages.

---

## 🌐 Live Demo
mallikarjunatravels.netlify.app

---

## 📌 Overview

Mallikarjuna Travels is a professional travel service platform built to deliver **safe, reliable, and comfortable transportation services** across Hyderabad and beyond.

This project focuses on:

* High-quality UI/UX
* Fast performance
* Scalable architecture
* Real-world business usability

---

## ✨ Features

* 🚗 **Outstation Taxi Services** – Long-distance travel across India
* ✈️ **Airport Transfers** – 24/7 pickup & drop services
* 🏙️ **City Sightseeing** – Explore Hyderabad landmarks
* 🛕 **Pilgrimage Packages** – Srisailam & temple tours
* 📱 **Fully Responsive Design** – Works on all devices
* ⚡ **Optimized Performance** – Fast loading & smooth experience
* 🎨 **Modern UI/UX** – Clean design with animations
* 🔐 **Secure & Scalable Structure**

## 🚀 Getting Started

### 1. Clone the Repository

```
git clone https://github.com/your-username/mallikarjuna-travels.git
cd mallikarjuna-travels
```

### 2. Install Dependencies

```
npm install
```

### 3. Run Development Server

```
npm run dev
```

Open 👉 http://localhost:3000
